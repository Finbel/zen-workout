## Common Use Cases

| Token | Use Case |
|-------|----------|
| `--space-1` (4px) | Tight inline spacing, icon gaps |
| `--space-2` (8px) | Default inline spacing, small component padding |
| `--space-3` (12px) | Medium component padding |
| `--space-4` (16px) | Standard component padding, form field spacing |
| `--space-6` (24px) | Section padding, card content |
| `--space-8` (32px) | Large section gaps |
| `--space-12` (48px) | Page section margins |
| `--space-16`+ | Major layout spacing |

---

## Usage

```css
/* Component internal spacing */
.button {
  padding: var(--space-2) var(--space-4);
  gap: var(--space-2);
}

/* Card layout */
.card {
  padding: var(--space-6);
}

.card-header {
  margin-bottom: var(--space-4);
}

/* Form layout */
.form-group {
  margin-bottom: var(--space-4);
}

.form-group label {
  margin-bottom: var(--space-1);
}

/* Section spacing */
.section {
  padding: var(--space-12) var(--space-6);
}
```
