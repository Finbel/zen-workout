# Typography

Our typography system uses CSS custom properties for consistent, scalable text styling
across the design system. As a **strongly opinionated** design system, we've made deliberate
choices about fonts, line heights, and letter spacing to create a calm, elegant reading experience
that embodies the Zen aesthetic.

## Line Height

<LineHeightSample name="Body" variable="--line-height-body" value="1.75" />

We use a single, generous line height (1.75) for all body text. This relaxed spacing creates
breathing room between lines, reducing visual density and creating a calm reading experience
that aligns with our Zen philosophy. The text should never feel cramped.

---

## Typography Principles

Our typography system follows specific principles to achieve the Zen aesthetic:

### Generous Leading (Line Height)

Zen is about breathing room. We use a relaxed line height of **1.75** for all body text.
This generous spacing ensures text never feels cramped, especially within Shoji grid cells.
The extra vertical space creates visual calm and improves readability.

### Letter Spacing for Headings

For headings, especially when using uppercase text, we apply slightly wide letter spacing
(`--letter-spacing-wide: 0.025em`). This adds a sense of elegance and luxury, making
headings feel more refined and intentional.

### Font Pairing

The combination of **Cormorant Garamond** (headings) and **Lato** (body) creates a
sophisticated contrast: the elegant serif draws attention to important content, while
the clean sans-serif ensures comfortable reading for longer passages.

## Usage

```css
/* Headings */
.heading-1 {
  font-family: var(--font-family-heading);
  font-size: var(--font-size-4xl);
  font-weight: var(--font-weight-bold);
  letter-spacing: var(--letter-spacing-wide);
}

.heading-2 {
  font-family: var(--font-family-heading);
  font-size: var(--font-size-3xl);
  font-weight: var(--font-weight-semibold);
  letter-spacing: var(--letter-spacing-wide);
}

/* Body text */
.body {
  font-family: var(--font-family-sans);
  font-size: var(--font-size-base);
  font-weight: var(--font-weight-regular);
  line-height: var(--line-height-body);
}

/* Code */
.code {
  font-family: var(--font-family-mono);
  font-size: var(--font-size-sm);
}
```
