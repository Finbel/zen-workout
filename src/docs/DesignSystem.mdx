## Philosophy

Zen Design is a React component library built on the principles of **calm**, **clarity**, and **intentionality**. Inspired by the minimalist aesthetics of Japanese design and the warmth of natural materials, this system prioritizes user focus and reduces visual noise.

## Design Principles

### 1. Embrace Simplicity

Remove the unnecessary so the necessary may speak. Components are designed with only essential features, avoiding bloat while maintaining flexibility.

### 2. Respect Whitespace

Space is not empty‚Äîit's breathing room. Generous spacing creates visual hierarchy and reduces cognitive load.

### 3. Use Nature's Palette

Our color system draws from the natural world: warm browns like aged wood, soft greens like morning sage, and cream tones like natural paper.

### 4. Subtle Interactions

Animations and transitions are gentle and purposeful. They guide without distracting.

## Getting Started

### Installation

```bash
npm install zen-design
# or
yarn add zen-design
```

### Basic Usage

```tsx
import { Button } from 'zen-design'
import 'zen-design/tokens.css'

function App() {
  return <Button variant="primary">Get Started</Button>
}
```

### Using Design Tokens

All design tokens are available as CSS custom properties:

```css
.my-component {
  background: var(--color-surface);
  padding: var(--space-4);
  border-radius: var(--radius-md);
  font-family: var(--font-family-sans);
}
```
