## Usage

Import the `Icon` component and use a typed `name` prop to display an icon:

```tsx
import { Icon } from 'zen-design'

// Basic usage
<Icon name="check" />

// With size
<Icon name="arrow-right" size={24} />

// With custom color
<Icon name="warning" color="var(--color-warning)" />

// With accessibility label
<Icon name="x" alt="Close" />
```

## Colors

Icons inherit `currentColor` by default, but can be colored using the `color` prop:

```tsx
<Icon name="info" color="var(--color-info)" />
<Icon name="check-circle" color="var(--color-success)" />
<Icon name="warning" color="var(--color-warning)" />
<Icon name="x-circle" color="var(--color-error)" />
```

## Accessibility

Use the `alt` prop to provide an accessible label for screen readers:

```tsx
// Decorative icon (hidden from screen readers)
<Icon name="arrow-right" />

// Meaningful icon (announced to screen readers)
<Icon name="x" alt="Close dialog" />
```

When an icon conveys meaning (like a close button), always provide an `alt` label.

## Adding New Icons

The Icon component includes 20 commonly used icons. To add more icons from Phosphor's library of 1,200+ icons:

### Step 1: Find the Icon

Browse [phosphoricons.com](https://phosphoricons.com/) to find the icon you need. Note the icon name (e.g., "Heart").

### Step 2: Update Icon.tsx

Open `src/components/Icon/Icon.tsx` and make these changes:

**1. Add the import:**

```tsx
import {
  // ... existing imports
  HeartIcon, // Add your new icon
} from '@phosphor-icons/react'
```

**2. Add to the IconName type:**

```tsx
export type IconName =
  | 'arrow-left'
  // ... existing names
  | 'heart' // Add kebab-case name
```

**3. Add to ICON_MAP:**

```tsx
const ICON_MAP: Record<IconName, PhosphorIcon> = {
  // ... existing mappings
  heart: HeartIcon, // Add the mapping
}
```

### Step 3: Use the Icon

```tsx
<Icon name="heart" size={24} color="var(--color-error)" />
```
